College Analytics Dashboard

2026 Internship Internal tool

A multi-part system for managing universities, student test results, and analytics dashboards with admin access and CSV import.

Overview

Built during an internship at RNMC — a practical tool for tracking college data and test outcomes. The goal was to replace scattered spreadsheets with something searchable, visual, and easier to maintain.

Features

  • Admin authentication and role-based access
  • CRUD for universities and test results
  • Dashboard with charts and summary stats
  • CSV import for bulk data uploads

Stack

  • React + TypeScript frontend
  • Node.js + Express API
  • SQLite database
  • JWT auth, bcrypt, multer for uploads
